# JavaFX Music Player

A simple and stylish music player built using JavaFX.

## Features
- Play, pause, stop, and skip songs.
- Volume control.
- Modern UI with CSS styling.
- Supports `.mp3` files.

## Requirements
- Java 21 or later.
- JavaFX SDK 21.
